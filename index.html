Смена времени модификации / создания файла в Linux


touch -a -m -t 201701011212.00 fileName.ext

где параметры:

-a = accessed (доступ)

-m = modified (модификация)

-t = timestamp (форматированное время: [[CC]YY]MMDDhhmm[.ss])
